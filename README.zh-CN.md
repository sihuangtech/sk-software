### 📢 讨论区分类

为了帮助我们高效地处理您的输入，请在发起新讨论时使用以下分类：

* **Bugs/Issues (错误/问题)：** 报告意外行为、错误或程序崩溃。
* **Feature Requests (功能请求)：** 建议新功能或改进现有功能。
* **Q&A (问与答)：** 提出关于如何使用我们软件的一般性问题。
* **Ideas (想法)：** 分享关于产品发展方向的总体思考和建议。
* **Announcements (公告)：** （由彩旗软件团队使用）用于发布官方新闻和更新。

### 📝 发帖前须知

1.  **先搜索：** 请在讨论区标签内使用搜索功能，查看您的问题或议题是否已被讨论过。
2.  **具体明确：** 在报告错误时，请务必包含您的操作系统、软件版本，以及重现问题的详细步骤。
3.  **保持尊重：** 在所有的交流中，请保持友好和建设性的语气。
